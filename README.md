### Étapes d'installation

1. Clonez ce dépôt :

   ```bash
   git clone <lien-du-dépôt>
   cd <nom-du-dossier>
   ```

2. Installez et démarrez chaque partie :

#### Frontend

```bash
cd frontend
npm install
npm start
```

#### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

> **Note :** Adaptez les commandes ci-dessus à votre projet si nécessaire.
